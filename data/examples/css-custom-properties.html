另一个卡片

CSS自定义属性（也称为CSS变量）允许我们定义可复用的值，并在样式表中引用它们。

CSS变量定义

:root {
  --primary-color: #3f51b5;
  --secondary-color: #ff4081;
  --text-color: #333333;
  --base-size: 1;
  --base-spacing: 1;
}

使用方式

.custom-button.primary {
  background-color: var(--primary-color);
  padding: calc(8px * var(--base-size));
  margin: calc(10px * var(--base-spacing));
}
