可访问性设计 Accessibility Design

确保所有用户都能访问和使用您的网站，无论其能力如何 Ensuring all users can access and use your website regardless of abilities

颜色对比度 Color Contrast

文本和背景之间必须有足够的对比度，使视力障碍用户能够阅读内容。 Text and background must have sufficient contrast to allow visually impaired users to read content.

良好对比度 Good Contrast
差的对比度 Poor Contrast

键盘导航 Keyboard Navigation

确保所有交互元素都可以通过键盘访问，并具有明显的焦点状态。 Ensure all interactive elements are keyboard accessible with visible focus states.

尝试使用Tab键在按钮之间导航 Try navigating between buttons using the Tab key

表单标签 Form Labels

所有表单控件必须有关联的标签，以帮助视障用户理解输入的内容。 All form controls must have associated labels to help visually impaired users understand what to input.

图像替代文本 Image Alt Text

为所有非装饰性图像提供替代文本，以便屏幕阅读器用户理解图像内容。 Provide alternative text for all non-decorative images so screen reader users can understand image content.

可调整文本大小 Resizable Text

文本应能放大至少200%而不丢失内容或功能，以帮助视力不佳的用户。 Text should be resizable up to 200% without loss of content or functionality to help users with low vision.

这段文本可以调整大小而不会破坏布局。 This text can be resized without breaking the layout.

ARIA属性 ARIA Attributes

使用ARIA属性增强复杂界面元素的可访问性。 Use ARIA attributes to enhance accessibility of complex interface elements.
